Otimização de hiperparâmetros¶

In [6]:
# Os dados so divididos em 3 (33% para validao) randomicamente e de forma balanceada,
# e o processo  repetido 'n_repeats' vezes.
cross_validator = RepeatedStratifiedKFold(n_splits=3, n_repeats=100, random_state=R_STATE)

for classifier, name, param in zip(classifiers, classifier_names, params):
    # Aqui, cada classificador  testado em cross-validation pelo RandomizedSearchCV,
    # para cada combinao randmica dos parmetros.
    grid = RandomizedSearchCV(classifier, param, scoring='accuracy',
                              cv=cross_validator, random_state=R_STATE).fit(X, y)

    # Seta parmetros achados no gridsearch
    classifier.set_params(**grid.best_params_)
    
    print('{0}: {1} - {2:.2%}'.format(name, grid.best_params_, grid.best_score_))

# Esta funo divide os dados randomicamente, na proporo definida por 'n_splits',
# treina os classificadores em cross-validation e repete o processo 'n_repeats' vezes,
# retornando as predies dos conjuntos de teste.
def massive_training(classifiers, X, y, n_splits=3, n_repeats=10):
    y_true = []
    y_preds = [[] for _ in classifiers]
    
    cv = RepeatedStratifiedKFold(n_splits=n_splits, n_repeats=n_repeats, random_state=R_STATE)
    
    for i_train, i_test in cv.split(X, y):
        X_train, X_test = X[i_train], X[i_test]
        y_train, y_test = y[i_train], y[i_test]

        y_true.extend(y_test)
        for j, classifier in enumerate(classifiers):
            classifier.fit(X_train, y_train)
            y_preds[j].extend(classifier.predict(X_test))
    
    return y_true, y_preds

y_true, y_preds = massive_training(classifiers, X, y, n_repeats=1000)

Conclusão¶

No geral, classificador que apresentou melhores resultados foi o Suport Vector Classifier. Abaixo est um resumo dos resultados por classe.

  • Iris-setosa: possvel notar que todos os classificadores desempenharam perfeitamente em todas as 50 mil predies. Houveram 5 falsos negativos em LgR, mas nenhum falso positivo. Ou seja, os trs classificadores garantem especificidade e preciso mximas para essa classe. A sensibilidade tambm pode ser considerada mxima para todos, se descartados os 5 falsos negativos de LgR.

  • Iris-versicolor: Para essa classe, temos um impasse. Apesar de aparentemente o classificador KNN ter acertado uma quantidade maior, ele tambm obteve a maior quantidade de falsos positivos. Ou seja, foi o que obteve menor preciso: 92.86%. Nesse aspecto, SVC teve melhor desempenho, com preciso de 96.05%.

  • Iris-virginica: Novamente os classificadores SVC e KNN possuem os melhores resultados, porm o SVC ligeiramente melhor na maioria das mtricas, s perdendo para o KNN na especificidade e preciso, mas ambas com uma diferena menor do que 1%.
